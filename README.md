# 🤖 TECHHOME Bot – Atención al Cliente

Bot de Telegram para **TECHHOME**, diseñado para gestionar solicitudes de clientes de forma automática y organizada.

Permite crear y consultar tickets de:
- 🛠 Servicio técnico
- 💰 Cotizaciones
- 📩 PQRS
- 🔍 Consulta de estado de tickets

La información se almacena en **Google Sheets** y se envían notificaciones por correo electrónico.

---

## 🚀 Funcionalidades

- Menú interactivo con botones
- Creación automática de tickets
- Generación de ID único por ticket
- Registro en Google Sheets
- Envío de correo al crear ticket
- Consulta del estado del ticket por ID
- Manejo de estados (Abierto, En proceso, Cerrado)
- Listo para ejecución 24/7 en Render

---

## 🧩 Tecnologías usadas

- Python 3.12
- python-telegram-bot
- Google Sheets API
- Gmail SMTP
- Render (deploy)
